Example taken from http://angg.twu.net/eev-current/anim/channels.anim.html
which inspired org-babel-screen.

* Listen on port 1234
  #+begin_src screen :session receiver :results silent
    netcat -l -p 1234
  #+end_src

* Send things to port 1234
  #+begin_src screen :session sender :results silent
    {
     echo hi
     slep 1
     echo bye
     sleep 1
    } | netcat -c localhost 1234
  #+end_src
